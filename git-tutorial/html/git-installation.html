Download Git

Go to https://git-scm.com/downloads or Google "Git Downloads" and download the recent stable version for your OS.

Install Git (Windows)

Follow installation wizard selections as shown in the images.

Use arrow keys to cycle the images, or click a thumbnail, or click the big picture.

Register to Github (optional)

At https://github.com/ in order to save your work on a remote server and share your work.

Github offers unlimited public repositories with unlimited collaborators for free.

Register to Bitbucket (optional)

At https://bitbucket.org in order to save your work on a remote server and share your work.

Bitbucket offers unlimited repositories (public and private) for free, up to 5 collaborators.
